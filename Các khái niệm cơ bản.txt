Cấu trúc 1 đoạn code trong css
selector{
	property: value;
}

* Selectors:
- Tag: p, div, h1, main, a, span
- Class: .demo, .header, .boy
- ID: #container, #footer
* Property: Các thuộc tính trong CSS
* Value: Giá trị của các thuộc tính
